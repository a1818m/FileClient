##这个项目是什么？
* 基于Qt4.7编写的文件传输系统中的客户端部分，能够实现远程计算机之间的文件传输

##这个项目的意义？
* 该项目是我用来练习的一个Demo。
* 你可以把它作为一个文件传输的工具(目前的版本操作性还并不友好，不推荐)。
* 你可以将它作为自己的练习项目，一起来添加完善相关功能。
